# Skill 9: Agentic Security

Patterns for guardrails, red teaming, and defense in depth.

## Status: Coming Soon

## Planned Examples

- **Input Guardrails**: Detecting prompt injection attempts
- **Output Guardrails**: Filtering PII and sensitive data
- **Red Teaming**: Using Garak and PyRIT for adversarial testing
- **Human-in-the-Loop**: Approval workflows for high-risk actions

## Key Concepts (Preview)

- Your agent is an attack surface
- Prompt injection is the new SQL injection
- Defense in depth - multiple layers of protection
- Regular red teaming finds vulnerabilities before attackers do
