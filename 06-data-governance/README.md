# Skill 6: Data Governance

Patterns for data quality, lineage tracking, and grounding.

## Planned Examples

- **Schema Validation**: Pydantic models for data quality gates
- **Lineage Tracking**: Recording data provenance
- **Strict Grounding**: Ensuring responses cite sources
- **Confidence Scoring**: Expressing uncertainty appropriately

## Key Concepts (Preview)

- Garbage in, hallucinations out - data quality is foundational
- Lineage tracking answers "where did this data come from?"
- Strict grounding prevents confident hallucinations
- Agents should express uncertainty when data is insufficient
